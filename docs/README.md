# Documentation

This directory contains the project documentation.

## Contents
- `PROMPTS.md`: Detailed breakdown of the prompts used in the extraction pipeline.
- `SECURITY.md`: Security architecture, threat model, and guardrails implementation.
- `OBSERVABILITY.md`: Observability strategy, metrics collected, and failure taxonomy.

## Context
This project implements a constrained structured extraction pipeline designed to pull nested JSON from noisy real-world text with >90% schema compliance, strictly using LLM structured outputs without regex fallbacks.
